# Vijay Sai Chigullapally — Portfolio

**Live:** https://portfolio-omega-six-j3zgm8e9ul.vercel.app/

A 3D interactive portfolio built with React, TypeScript, and Three.js.

## Features

- **3D AI Orb** — custom Three.js scene with wireframe icosahedron, orbital rings, and particle field
- **Animated Tech Stack** — CSS marquee showcasing 22+ technologies across two rows
- **Project Carousel** — 4 ML/AI projects with descriptions, tags, and GitHub links
- **Career Timeline** — work experience with animated timeline
- **Responsive** — mobile and desktop layouts

## Tech Stack

- React 18 + TypeScript + Vite
- Three.js (raw, no R3F)
- GSAP, Lenis
- CSS animations

## Projects Featured

| # | Project | Description |
|---|---------|-------------|
| 01 | F1 Spec RAG Agent | RAG pipeline using Google Gemini to query 2026 FIA F1 Technical Regulations |
| 02 | Fin Bot | Cross-platform Flutter & FastAPI personal finance chatbot with OCR & ML |
| 03 | Plant Disease Detector | CNN (MobileNetV2) trained on 50K+ images with crop recommender |
| 04 | Biomedical Classifier | Transfer learning (ResNet, VGG16) for X-ray & MRI classification |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Deployment

Deployed on Vercel with automatic redeploy on every `git push` to `main`.
